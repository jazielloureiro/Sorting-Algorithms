## Install

1. If you've git installed, type:

`$ git clone https://github.com/jazielloureiro/Sorting-Algorithms`

2. Go into the new directory, and type:

`$ make all`

Or, if you prefer to build a specific algorithm, you can:

`$ make bin/quick_sort`
